## 安装依赖

进入worktask 安装前端依赖

`npm install`

进入server 安装后端依赖

`cd server & npm install`

已经配置过淘宝镜像，无需配置

## 运行项目

进入server运行后端项目

`npm run dev`

根目录运行前端项目

`npm run start`

浏览器打开http://localhost:8080/work
