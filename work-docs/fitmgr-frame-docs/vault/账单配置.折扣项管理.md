# 主功能

* [x] 展示折扣项列表

* [x] 导出折扣项

* [x] 添加折扣类型说明

![](/assets/images/2021-01-07-09-45-39.png)

```html 
  折扣类型有系统，租户，Project
  <ul>
    <li>
      默认系统，不打折
    </li>
    <li>
      租户: 例如中国民航信息集团,生效范围是具体的租户名称
    </li>
    <li>
      Project: 例如业务管理系统,生效范围是具体的Project名称
    </li>
  </ul>
  ```
